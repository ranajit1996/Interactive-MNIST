Model Summary

        This Convolutional Neural Network (CNN) model was trained on 
        Google Colab GPU Runtime Environment on Classical MNIST Dataset
        of Handwritten Digits (60,000 training images and 10,000 testing
        images). The model had achieved 99.28% test accuracy with 
        0.022275074268919615% test loss. For better results Data 
        Augmentation was also applied.

        The Model is written using Tensorflow 2.1.0 Keras api. Model
